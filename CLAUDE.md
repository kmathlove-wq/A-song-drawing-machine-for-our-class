# Project Collaboration Rules

## Saving Tokens And Tool Calls

- Do not reread files that were already checked unless they may have changed.
- Do not make unnecessary tool calls.
- Run independent tool calls in parallel whenever possible.
- For long outputs, inspect only the needed range.
- Do not repeat details the user has already explained.

## Project Knowledge

- Add newly learned project knowledge to `AGENTS.md` or `CLAUDE.md` when useful.
- Keep both `AGENTS.md` and `CLAUDE.md` under 200 lines each.
- Do not revert existing changes unless the user explicitly asks for it.
